# WebTech Nextjs Lab

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
## API Routes
- / - default nextjs home page
- /about - about page
- /contacts - contacts page
- /articles - a list of articles (with some dummy data)
- /Article?articleId={id} - specific article
